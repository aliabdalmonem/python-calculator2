Simple calculator built with Python.

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🚫 Divide by zero protection

## ⚙️ Requirements

- Python 3

---

## 📥 Installation (Termux)

```bash
pkg update
pkg install git python -y

git clone https://github.com/aliabdalmonem/python-calculator2.git

cd python-calculator2

chmod +x install.sh

./install.sh

python main.py
```

---

## 🐧 Installation (Linux)

```bash
sudo apt update

sudo apt install git python3 -y

git clone https://github.com/aliabdalmonem/python-calculator2.git

cd python-calculator2

chmod +x install.sh

./install.sh

python3 main.py
```
